## Getting Started

1. Create a new BunPress site:
   ```bash
   bun create bunpress my-site
   ```

2. Start the development server:
   ```bash
   cd my-site
   bunpress dev
   ```

3. Build for production:
   ```bash
   bunpress build
   ```
